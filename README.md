# AI Resume & Portfolio Builder

## Project Description
This project generates a professional resume automatically using Python. 
The system collects user details and formats them into a structured resume. 
It also converts the resume into PDF format.

## Features
- Accepts user input
- Automatically formats resume
- Generates PDF file
- Beginner friendly implementation

## Technologies Used
- Python
- FPDF Library
- Google Colab

## How to Run
1. Open notebook in Google Colab
2. Run all cells
3. Enter your details
4. Download generated resume PDF
